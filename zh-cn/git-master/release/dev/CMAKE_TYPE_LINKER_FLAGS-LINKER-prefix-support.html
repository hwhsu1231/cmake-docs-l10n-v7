CMAKE_TYPE_LINKER_FLAGS-LINKER-prefix-support¶

  • The CMAKE_EXE_LINKER_FLAGS, CMAKE_EXE_LINKER_FLAGS_<CONFIG>, CMAKE_SHARED_LINKER_FLAGS, CMAKE_SHARED_LINKER_FLAGS_<CONFIG>, CMAKE_MODULE_LINKER_FLAGS, and CMAKE_MODULE_LINKER_FLAGS_<CONFIG> variables learned to support the LINKER: prefix.

    This support implies to parse and re-quote the content of these variables. This parsing is controlled by CMP0181 policy.
